# 📊 Data Analysis

A collection of fun and educational data analysis projects.

## Projects

### 🐍 Python

| Project | Description |
|---------|-------------|
| [Spurious Correlation Analysis](python/spurious_correlation_analysis/) | Exploring the fake correlation between pool drownings and Nicolas Cage films |
| [Divorce vs. Margarine](python/divorce_vs_margarine/) | The near-perfect (r=0.99) spurious correlation between Maine divorces and margarine consumption |
| [Chocolate vs. Nobel Prizes](python/chocolate_vs_nobel/) | Does eating chocolate make countries win Nobel Prizes? (r=0.88) |
| [Vegas Weddings vs. Polar Bears](python/vegas_weddings_vs_polar_bears/) | Seasonality as a confounder: weddings and polar bear activity (r=0.95) |
| [Chicken vs. Crude Oil](python/chicken_vs_oil/) | Price trend comparison and correlation exploration between chicken and crude oil |

## Getting Started

1. Clone this repo: `git clone https://github.com/GeronimoAnalytics/dataviz.git`
2. Navigate to `05_data_analysis/python/<project_name>`
3. Follow the project-specific README for setup instructions
